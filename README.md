Техническое задание: "Простой калькулятор"

1. Назначение

Консольное приложение для выполнения базовых арифметических операций над двумя числами.

2. Функциональные требования

2.1. Основной функционал

· Выполнение операций: сложение (+), вычитание (-), умножение (*), деление (/)
· Пошаговый ввод: первое число → операция → второе число
· Вывод результата в формате: число1 операция число2 = результат

2.2. Обработка ошибок

· Проверка корректности вводимых чисел
· Проверка допустимости операции (только +, -, *, /)
· Проверка деления на ноль
· Информативные сообщения об ошибках

3. Технические требования

· Язык: C#
· Платформа: .NET
· Тип: консольное приложение
· Архитектура: основной код в Main(), логика расчета в отдельном методе

4. Интерфейс пользователя

· Консольный ввод/вывод
· Четкие инструкции для пользователя
· Однократное выполнение расчета
· Завершение по нажатию любой клавиши

5. Критерии приемки

· Корректное выполнение всех арифметических операций
· Обработка всех типов ошибок ввода
· Удобство использования
· Стабильная работа без аварийных завершений
